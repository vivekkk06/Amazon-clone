# Amazon.com Clone

## Description
This project is a static clone of the Amazon.com homepage created using only HTML and CSS.  
It is built for learning and practice purposes, focusing on layout, design, and styling.

⚠️ Note: This project is **not responsive** and is best viewed on desktop screens.

## Features
- Amazon-like homepage layout
- Header, navigation bar, and product sections
- Clean UI using CSS
- Static design (no JavaScript)

## Technologies Used
- HTML5
- CSS3

## How to Run
1. Clone or download the repository
2. Open the `index.html` file in any web browser

## Project Structure
- `index.html` – Main HTML file  
- `style.css` – CSS styling file
